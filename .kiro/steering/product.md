# Product Overview

RetroWeb is a Chrome extension that transforms modern websites into nostalgic late 90s/early 2000s internet experiences. It applies retro visual effects including tiled backgrounds, custom cursors, vintage fonts, and neon styling to any webpage.

## Core Features

- 6 predefined retro themes:
  - **Geocities Chaos Mode** - Maximum 90s homepage chaos with rainbow text and ridge borders
  - **Neon Cyber 2001** - Matrix/hacker aesthetic with CRT scanlines and green glow
  - **Pixel Arcade** - 8-bit retro gaming with pixelated rendering
  - **VHS Analog Glitch** - VHS tracking lines and chromatic aberration
  - **Vaporwave A E S T H E T I C** - 80s/90s nostalgia with pink/purple/teal dreamscape
  - **Windows 95/98** - Classic gray panels with 3D beveled borders
- Theme-specific custom cursors
- Tiled background patterns
- Vintage fonts and text effects
- Customizable element toggles
- Per-domain settings support
- React-based popup UI for configuration
- Non-destructive overlay system (doesn't modify original page content)
- Dynamic content detection (modals, sidebars, dynamically loaded elements)

## Target Experience

The extension recreates the aesthetic of personal homepages from the late 90s/early 2000s web era, emphasizing maximalist design, animated elements, and playful visual chaos while maintaining usability.
